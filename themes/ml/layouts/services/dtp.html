Workflow

Our DTP process

A repeatable five-step process that delivers consistent, print-ready files every time.

1
Source File Analysis
We analyse your files for font availability, text expansion/contraction ratios, embedded graphics, and any language-specific issues such as RTL text direction or CJK character spacing before work begins.
2
Translation & Terminology
Content is extracted and translated by subject-matter linguists. We apply your existing glossaries and translation memories to ensure consistency with previous publications.
3
DTP Production
Translated text is flowed back into the native file. Our DTP operators resolve text overflow, adjust font sizes, reflow callouts and captions, and adapt graphics containing embedded text.
4
DTP Quality Review
A separate DTP QA specialist checks every page for overflow, orphaned lines, incorrect hyphenation, font substitution errors, colour profile consistency, and visual alignment with the source layout.
5
Delivery & Preflight
Files are packaged with all linked assets and fonts. Print-ready PDFs pass a standard preflight check. We deliver native source files alongside PDFs so you can make future edits independently.
